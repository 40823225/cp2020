Windows Networking(Windows網絡)

 英.From Vista to Windows 10, Windows networking is almost the same in every respect. To access the network settings, go to the control panel and click "Network and Internet", "Network and Sharing Center", or right-click "Open".
 中.從Vista到Windows 10，Windows網絡在各個方面都幾乎相同。 要訪問網絡設置，請轉到控制面板，然後依次單擊“網絡和Internet”，“網絡和共享中心”，或右鍵單擊“打開”。
 英.Select the Ethernet channel, then select properties, and then select TCP / IPv4.
 中.選擇以太網路，然後選擇屬性，然後選擇TCP / IPv4。
 英.Most networks are set up with DHCP, which will provide IP addresses, gateway and DNS server addresses for workstations.
Double-click the IPv4 Properties dialog box and set the two radio buttons to "Auto Obtain. The window on the right side of the figure below allows automatic allocation of APIPA addresses or allows users to configure static IPv4 addresses.
 中.大多數網絡都設置有DHCP，它將為工作站提供IP地址，網關和DNS服務器地址。
雙擊“ IPv4屬性”對話框，並將兩個單選按鈕設置為“自動獲取。”下圖右側的窗口允許自動分配APIPA地址或允許用戶配置靜態IPv4地址。
